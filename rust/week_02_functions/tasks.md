# Aufgaben

1. Schreibe eine Funktion `say_hello(name: &str) -> String`, die "Hello, {name}!" zurückgibt.
2. Schreibe eine Funktion `rectangle_area(width: f64, height: f64) -> f64`, die width * height berechnet.
3. Schreibe eine Funktion `is_positive(num: i32) -> bool`, die `true` zurückgibt, wenn `num > 0` ist.
4. Schreibe eine Funktion `max_num(a: i32, b: i32) -> i32`, die die größere der beiden Zahlen zurückgibt.
5. Schreibe eine Funktion `describe_temperature(temp: i32) -> String`, die "hot" für > 30, "warm" für > 15 und "cold" sonst zurückgibt.
