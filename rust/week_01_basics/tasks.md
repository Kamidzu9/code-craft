# Aufgaben

1. Schreibe eine Funktion `greet() -> String`, die "Hello, World!" zurückgibt.
2. Schreibe eine Funktion `create_name() -> String`, die eine Variable `name = "Rust"` erstellt und zurückgibt.
3. Schreibe eine Funktion `add(a: i32, b: i32) -> i32`, die zwei Zahlen addiert.
4. Schreibe eine Funktion `is_adult(age: i32) -> bool`, die `true` zurückgibt, wenn `age >= 18` ist.
5. Schreibe eine Funktion `introduce(name: &str, age: i32) -> String`, die "Hi, I'm {name} and I'm {age} years old!" zurückgibt.
