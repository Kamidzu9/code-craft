# Aufgaben

1. Schreibe eine Funktion `count_to(n: i32) -> Vec<i32>`, die einen Vektor mit Zahlen von 1 bis n zurückgibt.
2. Schreibe eine Funktion `sum_to(n: i32) -> i32`, die die Summe von 1 bis n berechnet (1+2+...+n).
3. Schreibe eine Funktion `countdown(n: i32) -> Vec<i32>`, die einen Vektor mit Zahlen von n bis 1 zurückgibt.
4. Schreibe eine Funktion `contains(arr: &[i32], val: i32) -> bool`, die prüft, ob `val` im Array vorkommt.
5. Schreibe eine Funktion `multiply_table(n: i32) -> Vec<i32>`, die [1*n, 2*n, ..., 10*n] zurückgibt.
