# Aufgaben

1. Schreibe eine Funktion `first_element(arr: &[i32]) -> Option<i32>`, die das erste Element zurückgibt oder `None`.
2. Schreibe eine Funktion `add_element(arr: &[i32], val: i32) -> Vec<i32>`, die einen neuen Vektor mit `val` am Ende zurückgibt.
3. Schreibe eine Funktion `double(arr: &[i32]) -> Vec<i32>`, die jedes Element mit 2 multipliziert.
4. Schreibe eine Funktion `find_max(arr: &[i32]) -> Option<i32>`, die das größte Element findet (mit Schleife).
5. Schreibe eine Funktion `even_numbers(arr: &[i32]) -> Vec<i32>`, die nur die geraden Zahlen zurückgibt.
