# Aufgaben

1. Schreibe eine Funktion `can_vote(age: i32) -> bool`, die `true` zurückgibt, wenn `age >= 18` ist.
2. Schreibe eine Funktion `grade(score: i32) -> String`, die Noten vergibt: 90+ = "A", 80+ = "B", 70+ = "C", 60+ = "D", sonst "F".
3. Schreibe eine Funktion `login(username: &str, password: &str) -> bool`, die "admin" und "secret" prüft.
4. Schreibe eine Funktion `absolute(n: i32) -> i32`, die den absoluten Wert (Betrag) zurückgibt.
5. Schreibe eine Funktion `animal_sound(animal: &str) -> String`, die mit `match` den Tierlaut zurückgibt: "dog" → "woof", "cat" → "meow", sonst "unknown".
