# Aufgaben - Woche 7: Fehlerbehandlung

1. Schreibe eine Funktion `safeDivide(a: number, b: number)`, die `a / b` zurückgibt oder `"Fehler: Division durch Null"` wenn `b === 0`.
2. Schreibe eine Funktion `parseAge(input: string)`, die den String in eine Zahl umwandelt oder `"Fehler: Keine gültige Zahl"` zurückgibt.
3. Schreibe eine Funktion `getFirstChar(s: string)`, die das erste Zeichen zurückgibt oder `null` bei leerem String.
4. Schreibe eine Funktion `getLengthSafe(s: string | null | undefined)`, die die Länge des Strings zurückgibt oder `0` bei `null`/`undefined`.
5. Schreibe eine Funktion `validateEmail(email: string)`, die `true` zurückgibt, wenn die E-Mail ein `@` enthält und mindestens 3 Zeichen lang ist.
