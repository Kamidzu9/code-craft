# Aufgaben - Woche 2: Funktionen

1. Schreibe eine Funktion `sayHello(name: string)`, die `"Hello, {name}!"` zurückgibt.
2. Schreibe eine Funktion `rectangleArea(width: number, height: number)`, die die Fläche eines Rechtecks berechnet (width × height).
3. Schreibe eine Funktion `isPositive(num: number)`, die `true` zurückgibt, wenn `num` größer als 0 ist.
4. Schreibe eine Funktion `max(a: number, b: number)`, die die größere der beiden Zahlen zurückgibt.
5. Schreibe eine Funktion `describeTemperature(temp: number)`, die `"hot"` zurückgibt wenn temp > 30, `"warm"` wenn temp > 15, und sonst `"cold"`.
