# Aufgaben - Woche 3: Bedingungen (Conditionals)

1. Schreibe eine Funktion `canVote(age: number)`, die `true` zurückgibt, wenn `age` größer oder gleich 18 ist.
2. Schreibe eine Funktion `grade(score: number)`, die eine Note zurückgibt: 90+ → "A", 80+ → "B", 70+ → "C", 60+ → "D", sonst "F".
3. Schreibe eine Funktion `login(username: string, password: string)`, die `true` zurückgibt, wenn `username` = "admin" UND `password` = "secret" ist.
4. Schreibe eine Funktion `absolute(n: number)`, die den absoluten Wert von `n` zurückgibt (ohne `Math.abs`).
5. Schreibe eine Funktion `animalSound(animal: string)`, die mit `switch` den Tierlaut zurückgibt: "dog" → "woof", "cat" → "meow", sonst "unknown".
