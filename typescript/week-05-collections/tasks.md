# Aufgaben - Woche 5: Arrays & Collections

1. Schreibe eine Funktion `firstElement(arr: number[])`, die das erste Element des Arrays zurückgibt, oder `null` wenn das Array leer ist.
2. Schreibe eine Funktion `addElement(arr: number[], val: number)`, die ein neues Array mit `val` am Ende zurückgibt.
3. Schreibe eine Funktion `double(arr: number[])`, die ein neues Array mit jedem Element × 2 zurückgibt.
4. Schreibe eine Funktion `findMax(arr: number[])`, die das größte Element findet (mit einer Schleife, nicht `Math.max`).
5. Schreibe eine Funktion `evenNumbers(arr: number[])`, die nur die geraden Zahlen behält.
