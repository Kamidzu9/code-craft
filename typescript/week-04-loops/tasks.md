# Aufgaben - Woche 4: Schleifen (Loops)

1. Schreibe eine Funktion `countTo(n: number)`, die ein Array `[1, 2, 3, ..., n]` zurückgibt.
2. Schreibe eine Funktion `sumTo(n: number)`, die die Summe `1 + 2 + ... + n` berechnet.
3. Schreibe eine Funktion `countdown(n: number)`, die ein Array `[n, ..., 2, 1]` zurückgibt.
4. Schreibe eine Funktion `contains(arr: number[], val: number)`, die `true` zurückgibt, wenn `val` im Array `arr` vorkommt.
5. Schreibe eine Funktion `multiplyTable(n: number)`, die ein Array `[1*n, 2*n, ..., 10*n]` zurückgibt.
