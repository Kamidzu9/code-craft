# Aufgaben - Woche 8: Mini Projekt - Todo Liste

1. Schreibe eine Funktion `createTodo(text: string)`, die ein Todo-Objekt `{ id, text, done }` zurückgibt. Die `id` soll eine fortlaufende Nummer sein (beginne bei 1).
2. Schreibe eine Funktion `addTodo(todos: object[], text: string)`, die ein neues Todo am Ende des Arrays hinzufügt und das neue Array zurückgibt.
3. Schreibe eine Funktion `listTodos(todos: object[])`, die nur die `text`-Felder aller Todos als String-Array zurückgibt.
4. Schreibe eine Funktion `markDone(todos: object[], id: number)`, die das Todo mit der gegebenen `id` auf `done: true` setzt.
5. Schreibe eine Funktion `removeTodo(todos: object[], id: number)`, die das Todo mit der gegebenen `id` entfernt.
