# greet() – Die erste Funktion

Eine Funktion ist ein wiederverwendbarer Code-Block. Mit `function` definierst du sie, mit `return` gibst du einen Wert zurück.

```typescript
function greet(): string {
  return "Hello, World!";
}
```

Der Rückgabetyp `: string` sagt TypeScript, dass die Funktion einen Text zurückgibt.
