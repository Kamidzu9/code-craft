# Aufgaben - Woche 1: Hello World & Variablen

1. Schreibe eine Funktion `greet()`, die den String `"Hello, World!"` zurückgibt.
2. Schreibe eine Funktion `createName()`, die eine Variable `name` mit dem Wert `"TypeScript"` erstellt und diese zurückgibt.
3. Schreibe eine Funktion `add(a: number, b: number)`, die zwei Zahlen addiert und das Ergebnis zurückgibt.
4. Schreibe eine Funktion `isAdult(age: number)`, die `true` zurückgibt, wenn `age` größer oder gleich 18 ist.
5. Schreibe eine Funktion `introduce(name: string, age: number)`, die den Satz `"Hi, I'm {name} and I'm {age} years old!"` zurückgibt.
