# Aufgaben - Woche 6: Objekte

1. Schreibe eine Funktion `createPerson(name: string, age: number)`, die ein Objekt `{ name, age }` zurückgibt.
2. Schreibe eine Funktion `greetPerson(person: { name: string })`, die `"Hello, {name}!"` zurückgibt.
3. Schreibe eine Funktion `birthday(person: { age: number })`, die ein neues Objekt mit `age` um 1 erhöht zurückgibt.
4. Schreibe eine Funktion `isSenior(person: { age: number })`, die `true` zurückgibt, wenn `age` >= 65 ist.
5. Schreibe eine Funktion `createTeam(name: string, members: string[])`, die ein Objekt `{ name, members, count }` zurückgibt.
