# Woche 3: Bedingungen – Aufgaben

1. Schreibe `canVote(age: Int) -> Bool`, das `true` zurückgibt, wenn `age >= 18`.
2. Schreibe `grade(score: Int) -> String`: 90+ → "A", 80+ → "B", 70+ → "C", 60+ → "D", sonst "F".
3. Schreibe `login(username: String, password: String) -> Bool`, das nur bei "admin"/"secret" `true` ist.
4. Schreibe `absolute(n: Int) -> Int`, das den absoluten Wert (Betrag) von `n` zurückgibt.
5. Schreibe `animalSound(animal: String) -> String` mit `switch`: "dog" → "woof", "cat" → "meow", default → "unknown".
