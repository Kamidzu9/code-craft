# Woche 8: Mini Projekt – Todo-Liste – Aufgaben

1. Schreibe `createTodo(text: String) -> [String: Any]`, das ein Dictionary mit `id` (Zufallszahl), `text` und `done: false` erstellt.
2. Schreibe `addTodo(todos: [[String: Any]], text: String) -> [[String: Any]]`, das ein neues Todo hinzufügt.
3. Schreibe `listTodos(todos: [[String: Any]]) -> [String]`, das nur die `text`-Felder zurückgibt.
4. Schreibe `markDone(todos: [[String: Any]], id: Int) -> [[String: Any]]`, das `done` auf `true` setzt.
5. Schreibe `removeTodo(todos: [[String: Any]], id: Int) -> [[String: Any]]`, das das Todo mit der gegebenen `id` entfernt.
