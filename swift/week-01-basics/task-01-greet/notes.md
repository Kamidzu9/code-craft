# Funktionen in Swift

Eine Funktion wird mit `func` definiert. Der Rückgabetyp steht nach `->`.

```swift
func greet() -> String {
    return "Hello, World!"
}
```

Der `return`-Schlüsselwort gibt den Wert zurück. Kurzform mit nur einem Ausdruck:

```swift
func greet() -> String { "Hello, World!" }
```
