# Woche 4: Schleifen – Aufgaben

1. Schreibe `countTo(n: Int) -> [Int]`, das ein Array `[1, 2, ..., n]` zurückgibt.
2. Schreibe `sumTo(n: Int) -> Int`, das `1 + 2 + ... + n` berechnet.
3. Schreibe `countdown(n: Int) -> [Int]`, das ein Array `[n, ..., 1]` (absteigend) zurückgibt.
4. Schreibe `contains(arr: [Int], val: Int) -> Bool`, das prüft, ob `val` in `arr` vorkommt.
5. Schreibe `multiplyTable(n: Int) -> [Int]`, das `[1*n, 2*n, ..., 10*n]` zurückgibt.
