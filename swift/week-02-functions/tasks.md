# Woche 2: Funktionen – Aufgaben

1. Schreibe `sayHello(name: String) -> String`, das "Hello, {name}!" zurückgibt.
2. Schreibe `rectangleArea(width: Double, height: Double) -> Double`, das `width * height` berechnet.
3. Schreibe `isPositive(num: Int) -> Bool`, das `true` zurückgibt, wenn `num > 0`.
4. Schreibe `maxNum(a: Int, b: Int) -> Int`, das die größere der beiden Zahlen zurückgibt.
5. Schreibe `describeTemperature(temp: Int) -> String`: "hot" wenn > 30, "warm" wenn > 15, sonst "cold".
