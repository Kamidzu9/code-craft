# Woche 5: Collections – Aufgaben

1. Schreibe `firstElement(arr: [Int]) -> Int?`, das das erste Element zurückgibt oder `nil` bei leerem Array.
2. Schreibe `addElement(arr: [Int], val: Int) -> [Int]`, das ein neues Array mit `val` am Ende zurückgibt.
3. Schreibe `double(arr: [Int]) -> [Int]`, das jedes Element mit 2 multipliziert.
4. Schreibe `findMax(arr: [Int]) -> Int?`, das das größte Element sucht (mit Schleife).
5. Schreibe `evenNumbers(arr: [Int]) -> [Int]`, das nur die geraden Zahlen zurückgibt.
