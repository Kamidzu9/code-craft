# Woche 7: Fehlerbehandlung – Aufgaben

1. Schreibe `safeDivide(a: Double, b: Double) -> String`, das `a/b` als String zurückgibt oder "Fehler" bei Division durch 0.
2. Schreibe `parseAge(input: String) -> String`, das die Zahl als String zurückgibt oder "Fehler" wenn keine gültige Zahl.
3. Schreibe `getFirstChar(s: String) -> Character?`, das das erste Zeichen zurückgibt oder `nil` bei leerem String.
4. Schreibe `getLengthSafe(s: String?) -> Int`, das die Länge zurückgibt oder 0 wenn `nil`.
5. Schreibe `validateEmail(email: String) -> Bool`, das prüft ob `"@"` enthalten ist und die Länge >= 3.
