# Aufgaben – Woche 7: Fehlerbehandlung

1. Schreibe `safe_divide(a, b)`, die `a / b` zurückgibt, oder `"Fehler: Division durch Null"` bei b == 0.
2. Schreibe `parse_age(input_str)`, die die Zahl zurückgibt, oder `"Fehler: Keine gültige Zahl"`.
3. Schreibe `get_first_char(s)`, die das erste Zeichen zurückgibt, oder `None` bei leerem String.
4. Schreibe `get_length_safe(s)`, die die Länge zurückgibt, oder `0` bei `None`.
5. Schreibe `validate_email(email)`, die prüft ob `"@"` enthalten ist UND mindestens 3 Zeichen lang.
