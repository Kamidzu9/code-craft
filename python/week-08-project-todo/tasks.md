# Aufgaben – Woche 8: Mini Projekt – Todo-Liste

1. Schreibe `create_todo(text)`, die `{"id": nächste_Zahl, "text": text, "done": False}` zurückgibt.
2. Schreibe `add_todo(todos, text)`, die eine neue Liste mit einem neuen Todo am Ende zurückgibt.
3. Schreibe `list_todos(todos)`, die nur die `text`-Werte als Liste zurückgibt.
4. Schreibe `mark_done(todos, id)`, die `done` auf `True` setzt (neue Liste zurück).
5. Schreibe `remove_todo(todos, id)`, die das Todo mit der angegebenen `id` entfernt.
