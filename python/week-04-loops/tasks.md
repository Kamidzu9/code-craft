# Aufgaben – Woche 4: Schleifen

1. Schreibe `count_to(n)`, die eine Liste `[1, 2, ..., n]` zurückgibt.
2. Schreibe `sum_to(n)`, die `1 + 2 + ... + n` berechnet und zurückgibt.
3. Schreibe `countdown(n)`, die eine Liste `[n, ..., 1]` zurückgibt.
4. Schreibe `contains(arr, val)`, die `True` zurückgibt wenn `val` in `arr` vorkommt.
5. Schreibe `multiply_table(n)`, die eine Liste `[1*n, 2*n, ..., 10*n]` zurückgibt.
