# Aufgaben – Woche 3: Bedingungen

1. Schreibe `can_vote(age)`, die `True` zurückgibt wenn `age >= 18`, sonst `False`.
2. Schreibe `grade(score)`, die `"A"` (score >= 90), `"B"` (>= 80), `"C"` (>= 70), `"D"` (>= 60) oder `"F"` zurückgibt.
3. Schreibe `login(username, password)`, die `True` zurückgibt wenn `username == "admin"` UND `password == "secret"`.
4. Schreibe `absolute(n)`, die den absoluten Wert von `n` zurückgibt (ohne `abs()` zu verwenden).
5. Schreibe `animal_sound(animal)`, die `"woof"` bei `"dog"`, `"meow"` bei `"cat"`, sonst `"unknown"` zurückgibt.
