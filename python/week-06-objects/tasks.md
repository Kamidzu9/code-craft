# Aufgaben – Woche 6: Dictionaries (Objekte)

1. Schreibe `create_person(name, age)`, die `{"name": name, "age": age}` zurückgibt.
2. Schreibe `greet_person(person)`, die `"Hello, {name}!"` zurückgibt.
3. Schreibe `birthday(person)`, die `age` um 1 erhöht und ein neues Dictionary zurückgibt.
4. Schreibe `is_senior(person)`, die `True` zurückgibt wenn `age >= 65`.
5. Schreibe `create_team(name, members)`, die `{"name": name, "members": members, "count": len(members)}` zurückgibt.
