# Aufgaben – Woche 5: Sammlungen (Listen)

1. Schreibe `first_element(arr)`, die das erste Element zurückgibt, oder `None` bei leerer Liste.
2. Schreibe `add_element(arr, val)`, die eine neue Liste mit `val` am Ende zurückgibt.
3. Schreibe `double(arr)`, die jedes Element mit 2 multipliziert und zurückgibt.
4. Schreibe `find_max(arr)`, die das größte Element findet (mit einer Schleife, nicht `max()`).
5. Schreibe `even_numbers(arr)`, die nur die geraden Zahlen behält und zurückgibt.
